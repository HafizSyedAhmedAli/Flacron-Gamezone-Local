**Project Overview**

Flacron GameZone is a football match discovery and live game platform developed for Flacron Enterprises LLC.

The platform allows users to:

Browse football leagues, teams, and matches

View live, upcoming, and finished matches

Access match detail pages

Subscribe to premium plans

View legal live streams or score-only pages

Read AI-generated match previews and summaries

This repository contains the full-stack codebase organized into separate frontend and backend applications.
